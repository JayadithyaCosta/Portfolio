# Chathuranga Costa Portfolio

Recruiter-focused multi-page portfolio built with Next.js + Tailwind using a premium dark glassmorphism UI.

## Run locally

```bash
npm install
npm run dev
```

## Routes

- `/`
- `/about`
- `/experience`
- `/projects`
- `/projects/[slug]`
- `/skills`
- `/contact`

## Notes

- Primary CTA is **Book a call**.
- Professional contact details only are displayed publicly.
- CV is available at `/cv/chathuranga-costa-cv.pdf`.
